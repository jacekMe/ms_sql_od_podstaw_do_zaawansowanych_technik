# MS SQL od podstaw do zaawansowanych technik

Repozytorium zawiera rozwiązania zadań realizowanych w trakcie kursu **"MS SQL od podstaw do zaawansowanych technik"**
na platformie [Udemy]. W kursie wykorzystywany jest Microsoft SQL Server i opiera się na podstawowych zapytaniach do
zaawansowanej optymalizacji i programowania w T-SQL.
Cały kurs opiera się na bazie danych **AdventureWorksLT2017**.

Kurs został stworzony przez Jakub Kozera i można znaleźć go pod tym linkiem (https://www.udemy.com/course/ms-sql-od-podstaw/?couponCode=MT150725B)


## Cel repozytorium

Chcę udokumentować moją ścieżkę nauki SQL i pokazać praktycznie zastosowania zdobytej wiedzy w formie uporządkowanego kodu, ćwiczeń i notatek.
To repozytorium może również posłużyć innym jako źródło przykładów i inspiracji do nauki języka SQL.
